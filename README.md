# Password Saver
A simple CLI password saver written in Python.

## 📌 Overview
Password Saver is a lightweight command-line Python application that is used for basic credential storage and simple access control.

## ⚙️ Features
- Master password protection for access control
- Store account credentials
- Retrieve saved data
- Simple and easy-to-use command-line interface

## 💻 How to use
### Menu options:
```
 1. Save
 2. Search
 3. View all
 4. Exit
Option: 
``` 
- Save: This option allows the user to store new login credentials.
- Search: This option looks for a specific account within the save data.
- View all: This option allows the user to see a list of all saved data.
- Exit: This option terminates the program.
  
## 📋 Requirements  
- Python 3.x
- No external libraries needed.
